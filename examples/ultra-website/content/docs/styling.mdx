# Styling

Did you know that browsers have a native way to style HTML documents? 

It's true, CSS is among the core languages of the open web and is standardized across many of the Web browsers we use today.

[Learn more about CSS](https://developer.mozilla.org/en-US/docs/Web/CSS)

---

If you would like to use an abstraction for CSS, **Ultra** has been known to support the following libraries:
